Instructions for the Google .NET Client API – GoogleApis.ServiceGenerator

The ServiceGenerator can be used to manually generate strongly typed libraries for services found in discovery itself, or which can be described by a discovery document. To see a full list of usages, run:

    ServiceGenerator.exe --help

Example usages:

  • ServiceGenerator.exe -c -a             Will generate and compile all discovery services.
  • ServiceGenerator.exe                     Will ask you for a service name and version, and will generate the service then.
